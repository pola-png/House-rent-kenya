## 🌐 Go Live (Optional)

### Deploy to Vercel:

1. Visit https://vercel.com
2. Click "New Project"
3. Import from GitHub
4. Add environment variables:
   ```
   NEXT_PUBLIC_SUPABASE_URL=your_url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key
   ```
5. Click "Deploy"

**Live in 2 minutes!** ⚡
